## Local usage

1. Open index.html in a browser, or serve locally (e.g., `npx serve .`).
2. Edit content in index.html, styling in css/style.css, and interactions in js/main.js.

## Deploy to GitHub Pages

1. Create a GitHub repository named `portfolio`.
2. Push this project to the `main` branch (root folder contains index.html).
3. In GitHub, open **Settings â†’ Pages** and select the `main` branch, `/ (root)` folder.
4. Wait for Pages to build; your site will be live at `https://satheeshkumart50.github.io/portfolio/`.

## Customization tips

- Update hero text, links, and project cards in index.html.
